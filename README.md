# Relatório Interativo sobre saneamento em Goiás desenvolvido em Python

Este trabalho é uma evolução e ramificação de dois projetos anteriores:
- Análise Descritiva e Preditiva da Gestão de Esgotos em Goiás
- Uso de Dados para Otimizar a Alocação de Recursos Públicos Destinados ao Saneamento Básico de Goiás

Mais informações: https://relatoriogoias.streamlit.app/

Recomenda-se que o relatório seja acessado por computador.
